В рамках это фичи мы реализуем возможность покупки генерации отчетов/проверки объявления о продаже автомобиля за деньги.

Если пользователь находитсья в `whitelist`, то он может делать проверку автомобилей без ограничений и платежей. Для остальных пользователй оплата необходима.

Чтобы заказать проверку автомобиля, пользователь должен иметь кредиты на своем аккаунте. Стоимость 1 проверки - 1 кредит. Пользователь может приобрести кредиты через платежную систему, которая будет интегрирована в приложение. На данный момент приложение интегрированно только с платежной системой `Lemon Squeezy`.

Пользователь может приобрести кредиты через интерфейс приложения, выбрав нужное количество кредитов и оплатив их через `Lemon Squeezy`. После успешной оплаты, кредиты будут добавлены на аккаунт пользователя, и он сможет использовать их для заказа проверок автомобилей.

Приложение предоставляет 2 пакета кредитов для покупки:
1. 1 кредит - 1 проверка - 20 PLN
2. 3 кредита - 3 проверки - 40 PLN

**ВАЖНО**: кредиты не возвращаются и не обмениваются на деньги. Кредиты списываются только когда запрос пользователя успешно обработан и результат проверки предоставлен. Если запрос не может быть обработан (например, из-за технической ошибки), кредиты не будут списаны, и пользователь сможет повторить попытку без дополнительных затрат.

Авторизованный пользователь видит свой баланс внизу страницы (уже реализовано). Рядом с этой областью должна быть кнопка "Пополнить баланс", которая открывает интерфейс для покупки кредитов. После успешной оплаты, баланс пользователя обновляется, и он может использовать кредиты для заказа проверок автомобилей.
